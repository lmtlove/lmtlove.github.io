react hook函数

Created2022-08-26Updated2022-08-26

useEffect

介绍：第一个参数是函数，第二个参数是依赖项，可不传

  1. useEffect进入页面执行一次
  2. useEffect没有依赖项则执行回调函数里面的
  3. useEffect没有依赖项进入页面只执行一次
  4. useEffect有依赖项则依赖项更新useEffect也会执行

memo

介绍:主要是使用子组件使用 父组件传给子组件props参数
父组件的值修改 子组件也会重新渲染
